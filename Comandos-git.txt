git --version -> mostra a versao do git

git init -> cria um repositorio git vazio na sua maquina

git add . -> envia todos os arquivos da pasta para o pré stage

git status -> mostra os arquivos que estao no pré stage

git config --global user.email uilhamgoncalves@gmail.com

git config --glogal user.name uilham-max

git commit -m "primeiro commit" -> envia para o stage

git branch -M "main"

git remote add origin https://github.com/uilham-max/ProjetoEscolaPoo.git

git push -u origin main

------------------------------------

git add . 				-> pré stage
git status 				-> visualiza arquivos no pré stage
git commit -m "mensagem" 	-> empurra para o stage
git push -u origin main 	-> envia do stage para nuvem
